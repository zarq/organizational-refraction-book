# Organizational Refraction — Book Repository

*Why Your Strategy Arrives Somewhere It Was Never Aimed*

This repository holds all working content for the book on organizational refraction: chapter drafts, research case studies, and the master outline. Website and marketing content lives in the `/website/` directory (managed by CMO/WebsiteMarketingManager).

## Repository Structure

```
book/
  outline/          — Master outline with chapter synopses and research-gap register
  frontmatter/      — Title page, copyright, table of contents
  chapters/         — Chapter drafts in progress
  research/
    case-studies/   — Primary and secondary case write-ups
    positive-correction/   — Refraction-resistant and positive-correction cases
    interventions/  — Intervention efficacy evidence
    expert-interviews/     — Synthesized expert interview findings
website/            — Website, marketing, and positioning content (CMO)
  speaking/         — Speaking engagement copy and bios
  (landing pages, hero copy, launch sequences, press kit, CMS content files)
README.md           — This file
```

## Commit Practice

Any completed chapter draft, outline update, or research document **must be committed and pushed before the producing task is marked done**. This is non-negotiable — it keeps the pipeline visible and protects work from being lost in closed Paperclip issues.

When committing book content:
- Use descriptive commit messages: `book: add Chapter 1 draft v1`
- One logical unit per commit (one chapter, one case study, one outline revision)
- Add `Co-Authored-By: Paperclip <noreply@paperclip.ing>` to commit messages
